# Laboratory work #6 Tree structure

1. General task.\
   Realize a binary ordered tree for storing integers (values are not repeated) and perform standard actions with it:
   filling, searching, deleting and printing.

   **Requirements**:
   Organize work with the tree in menu format with the following functionalities:
    1. Add node: Add a new value entered by the user. In case such a value already exists - inform the user about it and
       do not add it.
    2. Find node: Find the specified node of its parent (if any), its descendants (if any). Output all information about
       it.
    3. Delete node: Delete the node specified by value.
    4. Print tree: Print all the nodes of the tree with different traversals: forward, backward, symmetric and width.
    5. Output

2. Individual task.
   For the tree from the first task perform your own variant of the individual task: Find the value of the largest
   element of the tree.

# Лабораторная работа №6 Структура дерево

1. Общее задание.\
   Реализовать бинарное упорядоченное дерево для хранения целых числа (значения не
   повторяются) и выполнить с ним стандартные действия: заполнение, поиск, удаление и печать.

   **Требования**:
   Организовать работу с деревом в формате меню со следующими функционалами:
    1. Добавить узел: Добавить новое значение, введенное пользователем. В случае, если такое значение уже
       существует - сообщить об этом пользователю и не добавлять.
    2. Найти узел: Найти указанный по значению узел его родителя (если имеется), его потомков (если
       имеются). Вывести всю информацию о нем.
    3. Удалить узел: Удалить указанный по значению узел.
    4. Печать дерева: Вывод всех узлов дерева разными видами обходов: прямым, обратным, симметричным и
       в ширину.
    5. Выход

2. Индивидуальное задание.\
   Для дерева из первого задания выполнить свой вариант индивидуального задания: Найти величину наибольшего элемента
   дерева.
